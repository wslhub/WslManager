**WSL Manager**는 Windows 10(2004 업데이트 이상)과 Windows Server 2004에서 WSL(Windows Subsystem for Linux) Distro 관리를 효율적으로 하실 수 있도록 해 주는 도구입니다.

이 프로그램은 WSL을 이용하여 여러 Linux 배포판들을 설치하고 관리할 수 있도록 해 줍니다. 이 프로그램은 현재 개발 중이며 첫 번째 기능을 완료하는 데에 시일이 소요될 수 있음을 양해 부탁드립니다.

## 기능 요약 설명

* GUI 기반의 인터페이스로 Linux Distro를 손쉽게 실행합니다.
* Linux Distro를 직접 열 수 있습니다.
* 직접 생성한 시스템 tarball 아카이브를 WSL(Windows Subsystem for Linux)로 가져올 수 있습니다.
* 사용 중인 Distro를 tarball 아카이브로 내보낼 수 있습니다.
* Distro 내에 명령이나 스크립트를 직접 보내 실행할 수 있습니다.
* wsl.exe의 기본 Distro를 설정합니다.
* 특정 사용자의 자격증명을 이용하여 WSL Distro를 실행합니다.

## FAQ (자주 묻는 질문)

### 이 도구의 기능은 wsl.exe에서도 제공하는 듯 한데, 이 도구를 사용해야 하는 이유가 있나요?

WslManager는 WSL의 Distro 관리 기능을 GUI 방식으로 사용하실 수 있도록 설계하였습니다. 이 도구를 사용하시면 시스템에서 여러 WSL Distro들을 사용하실 때 직관적으로 관리할 수 있으므로 추가적인 생산성이 향상될 걸로 기대합니다.

직접 명령어를 입력하시는 데에 익숙하시다면 그냥 wsl.exe을 이용하여 직접 호출하셔도 무방합니다. 동일한 기능입니다.
